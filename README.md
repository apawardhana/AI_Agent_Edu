### Installation

1. **Clone repository**
```
git clone https://github.com/apawardhana/AI_Agent_Edu.git
cd AI_Agent_Edu
```
2. **Install Dependencies**
```
cd backend/python_app
pip install requirements.txt
uvicorn main:app --reload
```

3. **Setup Env**
```
cd frontend
npm run dev
```

4. **Edit Agent**
   ```
cd frontend/src/components/agents
   ```
